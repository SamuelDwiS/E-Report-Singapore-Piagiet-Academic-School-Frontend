# E-Report-Singapore Piaget Academy
E-Report Singapore Piaget Academy adalah platform manajemen laporan nilai siswa digital yang dibangun menggunakan Next.js 16 dan Tailwind CSS. Aplikasi ini dirancang untuk memfasilitasi sekolah dalam mengelola data akademik, penilaian, dan distribusi rapor secara efisien, aman, dan terstruktur.

Dengan menggunakan template TailAdmin, sistem ini menyediakan antarmuka dasbor yang komprehensif bagi admin dan guru untuk memantau perkembangan siswa secara real-time melalui panel kontrol yang intuitif dan responsif.

## Fitur Utama
* Manajemen Data Akademik: Kelola data siswa, guru, dan mata pelajaran dalam satu platform terpusat.

* Sistem Penilaian Digital: Input dan kalkulasi nilai rapor yang mendukung berbagai komponen penilaian sesuai kurikulum sekolah.

* Dasbor Informatif: Visualisasi data statistik performa siswa menggunakan komponen UI modern dari TailAdmin.

* E-Report Generation: Pembuatan dokumen rapor digital yang siap diunduh atau dicetak secara otomatis.

## Teknologi yang Digunakan
Proyek ini memanfaatkan cutting-edge stack untuk memastikan performa dan skalabilitas tinggi:

* Next.js 16: Memanfaatkan fitur Server-Side Rendering (SSR) dan Static Site Generation (SSG) untuk akses data rapor yang cepat.

* React 19: Memberikan pengalaman pengguna yang mulus dengan manajemen state yang efisien.

* Tailwind CSS: Memastikan tampilan dasbor yang modern, bersih, dan sepenuhnya responsif di berbagai perangkat.

* TypeScript: Meningkatkan keamanan kode dan mempermudah pemeliharaan jangka panjang melalui static typing.
